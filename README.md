## 实验结论

| 冗余度 | 最少磁盘/FG | 支持failgroup | 混合大小磁盘 |
|--------|-------------|---------------|--------------|
| EXTERNAL | 1个磁盘 | ❌ 不支持 | ✅ 支持 |
| NORMAL | 2个FG | ✅ 支持 | ❌ 不支持 |
| HIGH | 3个FG | ✅ 支持 | ❌ 不支持 |

## 关键发现

1. **EXTERNAL冗余度**不支持指定failgroup（ORA-15067）
2. **NORMAL/HIGH冗余度**不支持混合不同大小磁盘（ORA-15410）
3. **NORMAL冗余度**要求各failgroup磁盘数量一致（ORA-15411）
4. **HIGH冗余度**至少需要3个failgroup（ORA-15167）
